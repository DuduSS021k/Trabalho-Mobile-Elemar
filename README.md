Este projeto consiste em um aplicativo móvel desenvolvido como parte da disciplina de Desenvolvimento Mobile. O objetivo principal é oferecer uma interface intuitiva onde os usuários podem visualizar, salvar e gerenciar frases motivacionais. O projeto foi construído utilizando as seguintes tecnologias:
• React Native: Framework para desenvolvimento cross-platform.
• TypeScript: Adição de tipagem estática para maior segurança e manutenção do código.
• AsyncStorage: Sistema de armazenamento local para persistência de dados.
• Expo: Conjunto de ferramentas e serviços para facilitar o desenvolvimento e teste.
